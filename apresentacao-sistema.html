CampanhaFácil 01. Visão Geral & Arquitetura

Introdução ao Sistema

O CampanhaFácil é uma solução completa para gerenciamento e venda de bilhetes de sorteios e prêmios. O sistema oferece uma interface robusta para administradores gerenciarem suas campanhas e uma interface intuitiva para clientes realizarem compras de números via PIX de forma rápida e segura.

Arquitetura Tecnológica

A aplicação foi construída utilizando modernas tecnologias de desenvolvimento web, garantindo alta performance, responsividade e estabilidade:

Frontend Moderno

Interface responsiva baseada em React estruturada com roteamento dinâmico via TanStack Router para navegação suave e instantânea.

React TypeScript TanStack Router

Visualizações & Mídia

Gráficos interativos em tempo real via Recharts e gerador de vídeos MP4 nativo usando Canvas API para compartilhamento rápido nas redes.

Recharts Canvas API MediaRecorder

CampanhaFácil 02. Funcionalidades Core

Recursos Principais do Painel Admin

Dashboard Financeiro e de Vendas

O painel administrativo consolida dados essenciais para o gestor em tempo real:

  • Métricas Gerais: Total arrecadado, campanhas ativas, campanhas encerradas e total de bilhetes vendidos.
  • Gráfico Comparativo de Vendas: Visualização nítida comparando a receita gerada em Balcão (vendas físicas presenciais) vs. Online.
  • Gestão de Pacotes Promocionais: Editor dinâmico e flexível para criar descontos progressivos por quantidade de bilhetes comprados.

Experiência de Sorteio Automatizada

O módulo de sorteio simula a escolha do ganhador com alto impacto visual:

  • Modo Apresentação: Tela cheia otimizada para projetores ou transmissões ao vivo (Live streaming).
  • Geração Instantânea de Vídeos: O sistema compila automaticamente um vídeo da animação do sorteio com a foto do prêmio e dados do ganhador para exportação e postagem instantânea (Instagram/WhatsApp).

CampanhaFácil 03. Fluxo de Operação

Fluxo de Compra e Sorteio

O processo foi otimizado para minimizar fricção e garantir a máxima taxa de conversão:

1. Criação da Campanha: O administrador cadastra o prêmio, fotos, quantidade de números e define pacotes de desconto promocionais.

2. Venda Multicanal: Vendas são feitas online pelo próprio cliente ou fisicamente pelo operador de balcão (gerando o PIX Copia e Cola instantaneamente).

3. Encerramento & Sorteio: Atingido o prazo, o administrador realiza o sorteio na plataforma com animações 3D de contagem regressiva e confetes.

4. Compartilhamento do Resultado: O sistema gera o vídeo ou banner em alta resolução e o administrador compartilha diretamente.

Vantagens da Solução

Operação Otimizada

Gestão unificada que reduz erros de contabilidade entre bilhetes físicos vendidos na loja e vendas online.

Marketing Eficiente

Exportação automatizada de mídia poupa tempo com design de posts e agiliza a divulgação dos vencedores.
